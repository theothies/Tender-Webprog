# Tender
## Was ist Tender?
Projekt zur Vorlesung Webporgrammierung an der DHBW Karlsruhe
Tender ist eine Single Page App, auf der man das Rind, dessen Fleisch man kaufen
will auf gute alte "Social Media Manier" kennenlernen kann.
